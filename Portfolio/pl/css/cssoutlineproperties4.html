Outline - Shorthand property

Longhand 
      div.longhand {
        border: 1px solid black;
        outline-width: 5px;
        outline-style: dotted;
        outline-color: red;
      }


Shorthand 
      div.shorthand {
        border: 1px solid black;
        outline: 5px dotted red;
      }

Note: IE8 supports the outline properties only if a !DOCTYPE is specified.
