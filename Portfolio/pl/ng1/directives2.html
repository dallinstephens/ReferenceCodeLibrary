Directives: 2

Directives start with the prefix ng-. The following three are directives:
(1) ng-app - initializes an AngularJS application,
(2) ng-init - initiates application data, and
(3) ng-model - binds the value of HTML controls (such as input, select, and textarea) to application data.

Note: The above explanation was retrieved from w3schools.com.

<div ng-app="" ng-init="vegetableName='asparagus'">

  <p><b>Try changing the name of the vegetable in the input box.</b></p>

  <p>Vegetable: <input type="text" ng-model="vegetableName"></p>

  <p style="color:red;">The vegetable is {{vegetableName}}.</p>

</div>
