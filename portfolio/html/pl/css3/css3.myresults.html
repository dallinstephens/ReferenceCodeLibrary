CSS3 border-radius Property

The border-radius property allows you to add rounded corners to elements.

Rounded corners for an element with a specified background color:

        #rounded-corners-example-1 {
          border-radius: 30px;
          background: #ffb3b3;
          width: 300px;
          height: 150px;
          padding: 20px;
        }

Rounded corners for an element with a border:

        #rounded-corners-example-2 {
          border-radius: 30px;
          border: 5px solid #ffb3b3;
          width: 300px;
          height: 150px;
          padding: 20px;
        }

Rounded corners for an element with a background image:

        #rounded-corners-example-3 {
          border-radius: 30px;
          background: url(images/paper.gif);
          background-position: left top;
          background-repeat: repeat;
          width: 300px;
          height: 150px;
          padding: 20px;
        }

CSS3 border-radius - Specify Each Corner

The border-radius property allows you to add rounded corners to elements.

Four values: first value applies to top-left, second value applies to top-right, third value applies to bottom-right, and fourth value applies to bottom-left corner

        #rounded-corners-example-4 {
          border-radius: 20px 80px 40px 5px;
          background: #ff4d4d;
          color: #FFF;
          width: 350px;
          height: 150px;
          padding: 20px;
        }

Three values: first value applies to top-left, second value applies to top-right and bottom-left, and third value applies to bottom-right

        #rounded-corners-example-5 {
          border-radius: 5px 80px 30px;
          background: #00bfff;
          width: 350px;
          height: 150px;
          padding: 20px;
        }

Two values: first value applies to top-left and bottom-right corner, and the second value applies to top-right and bottom-left corner

        #rounded-corners-example-6 {
          border-radius: 8px 200px;
          background: #00ff80;
          width: 350px;
          height: 150px;
          padding: 20px;
        }
