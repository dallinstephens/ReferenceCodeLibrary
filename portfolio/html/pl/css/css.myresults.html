Border Styles

Dotted - Defines a dotted border

Dashed - Defines a dashed border

Solid - Defines a solid border

Double - Defines a double border

Groove - Defines a 3D grooved border. The effect depends on the border-color value

Ridge - Defines a 3D ridged border. The effect depends on the border-color value

Inset - Defines a 3D inset border. The effect depends on the border-color value

Outset - Defines a 3D outset border. The effect depends on the border-color value

None - Defines no border

Hidden - Defines a hidden border

Mix - Defines a combination of borders

Border Width

Dotted (border-width: thin;) - Defines a dotted border

Dashed (border-width: medium;) - Defines a dashed border

Solid (border-width: thick;) - Defines a solid border

Double (border-width: 1px;) - Defines a double border

Groove (border-width: 2px;) - Defines a 3D grooved border. The effect depends on the border-color value

Ridge (border-width: 4px;) - Defines a 3D ridged border. The effect depends on the border-color value

Inset (border-width: 8px;) - Defines a 3D inset border. The effect depends on the border-color value

Outset (border-width: 16px;) - Defines a 3D outset border. The effect depends on the border-color value

None - Defines no border

Hidden - Defines a hidden border

Mix (border-width: 2px[top] 10px[right] 4px[bottom] 20px[left];) - Defines a combination of borders
